Job Tracker Application

A Job Tracker web application built using Python and Django that helps users manage and track their job applications in an organized manner. The application allows users to store job details, monitor application status, and keep notes related to their job search process.

Features

* User Authentication (Login / Logout)
* Add and Manage Companies
* Add, Edit, and Delete Job Applications
* Track Application Status 
* Simple and user-friendly interface

Tech Stack

Backend: Python, Django
Frontend: HTML, CSS, Bootstrap
Database: SQLite
